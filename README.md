## Build Setup 

``` 
 # clone
git clone https://github.com/Mangomilkhh/Hackers.git
```

```
# install dependencies
yarn install
```

```
# Compiles and hot-reloads for development
yarn dev
```

```
# Compiles and minifies for production
yarn build
